Nosferatu core files
===============

Part of the [Nosferatu](https://github.com/Nosferatu-lib) library.

## Installation

You will need

 * [Node.js](http://nodejs.org/)
 * [Bower](http://bower.io/)
```
$ npm install -g bower
```

Then install this component with

```
$ bower install nosferatu-core --save
```
